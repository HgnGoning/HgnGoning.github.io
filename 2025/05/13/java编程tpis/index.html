数组排序：Arrays.sort

Arrays.sort(int[] a, int fromIndex, int toIndex)

这种形式是对数组部分排序，也就是对数组a的下标从fromIndex到toIndex-1的元素排序，注意：下标为toIndex的元素不参与排序哦！

第二个参数是 Comparator，定义元素的排序规则。

可以直接使用Lambda

(p, q) -> p[0] - q[0]

  • 返回负值：p 应排在 q 前（左端点更小）。
  • 返回正值：q 应排在 p 前（左端点更大）。
  • 返回0：两者视为相等。
