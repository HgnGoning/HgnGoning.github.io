## git

### git 推送因合并冲突被拒绝

**. 本地缓存未刷新**

虽然 `git pull` 显示最新，但可能本地缓存未正确更新。尝试：

```bash
# 强制拉取远程分支覆盖本地（谨慎使用，会丢弃本地未提交修改）
git fetch --all
git reset --hard origin/hgn
```

然后重新提交你的修改并推送：

```bash
git add src/META-INF/ #添加到暂存区
git commit -m "添加 META-INF 目录"
git push origin hgn
```

### git提交文件过大，暂存区无法提交

git查看历史提交

```bash
git log --oneline
```

重置分支到目标提交, abc1234是上一个命令的id

```bash
git reset --hard abc1234  # 丢弃此提交之后的所有历史
```
